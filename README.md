# Starter Kit

- Simple and lightweight boilerplates
- Launch local server and live reload
- Automatically code format

## Technologies

### simple-web

- HTML: EJS
- JS: ES2015 (Webpack + Babel)
- CSS: PostCSS (import, cssnext, cssnano)

## Install

```
$ sh install.sh directory-you-want-to-copy
```

## Usage

### Setup

```
$ yarn install
```

### Develop

```
$ yarn start
```

### Production build

```
$ yarn build
```

### Format

```
$ yarn format
```
